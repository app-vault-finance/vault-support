Politique de confidentialité

Vos données restent locales d’abord.

Vault est conçu pour conserver en priorité vos données financières sur vos appareils. Nous n’exigeons pas de compte Vault séparé pour utiliser l’application.

Ce que Vault stocke

Les comptes, transactions, budgets, objectifs, catégories, éléments récurrents et pièces jointes de reçus sont stockés dans les données locales de l’application sur votre appareil.

Synchronisation iCloud

Si vous activez la synchronisation avec Vault Pro, vos données sont synchronisées de façon privée via votre propre compte iCloud avec CloudKit. Vault n’exploite pas de backend financier personnel.

Achats

Les abonnements et achats uniques sont traités par Apple via l’App Store. Vault utilise uniquement l’état d’achat nécessaire pour débloquer les fonctionnalités Pro.

Contact assistance

Si vous contactez l’assistance par e-mail, votre message et les pièces jointes envoyées sont utilisés uniquement pour enquêter sur votre demande et y répondre.
